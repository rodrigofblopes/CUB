## 🌐 Próximos Passos - Ativar GitHub Pages

Para deixar sua calculadora online e acessível pelo navegador:

### 1. Acesse seu Repositório no GitHub
```
https://github.com/rodrigofblopes/CUB
```

### 2. Configure o GitHub Pages
1. Clique na aba **"Settings"** (Configurações)
2. No menu lateral esquerdo, clique em **"Pages"** 
3. Em **"Source"**, selecione **"Deploy from a branch"**
4. Em **"Branch"**, selecione **"main"**
5. Em **"Folder"**, deixe **"/ (root)"**
6. Clique em **"Save"**

### 3. Aguarde o Deploy (1-2 minutos)
O GitHub processará os arquivos e criará o site automaticamente.

### 4. Acesse sua Calculadora Online
Após alguns minutos, sua calculadora estará disponível em:
```
https://rodrigofblopes.github.io/CUB/
```
